Build a Github blog with Hexo

使用 Hexo 搭建 Github 博客

你也可以使用 Hexo 官方文档

Step1: New repository

  1. 在你的Github上新建一个库，名字严格遵守为：.github.io。
  2. setting你的库,选择master branch, 当然你也可以点击 Chose theme 选择简单的博客主题。
  3. 在浏览器中打开 .github.io ，发现库里面的内容已经暴露在外网了。

Step2: Install Git, node.js, Hexo

Install Git
  • Windows: Download & install git.

  • Mac: Install it with Homebrew, MacPorts or installer.

  • Linux (Ubuntu, Debian): sudo apt-get install git-core

  • Linux (Fedora, Red Hat, CentOS): sudo yum install git-core
